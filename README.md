# EDA on Retail Sales Data

This project is completed as part of the Data Analytics Internship at Oasis Infobyte.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Objective
To perform Exploratory Data Analysis and extract insights from retail customer data.

## Key Insights
- Spending patterns vary by day
- Higher bills result in higher tips
- Clear correlation between bill amount and tips

## Conclusion
EDA helps businesses understand customer behavior and make data-driven decisions.
